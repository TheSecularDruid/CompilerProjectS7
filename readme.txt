Nous vous présentons le projet de compilation réalisé par
Raphaël G. et Maëlle Toy-Riont--Le Dosseur, disponible dans le dossier rendu. 

Voici les points réalisés durant le projet :
- mécanisme de déclarations explicite de variables,
- expresssion arithmétiques arbitraire de type calculatrice,
- lectures ou écritures mémoires via des affectations avec variable utilisateur,
- mécanisme de typage comprenant notamment int et float,
- affichage des opérations booléennes

Nous avons aussi géré des cas particuliers et des cas d'erreurs :
- erreur dans le cas d'une affectation à une variable non définie
- erreur si utilisation d'une variable déclarée mais non initialisée
- cast implicite dans le cas d'une expression arithmétique entre un int et un float
- warning pour le cast d'un int en float
